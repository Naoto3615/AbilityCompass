# ステップアップナビ 要件定義書

## アプリ概要

**ステップアップナビ**は、軽度知的障害を持つ大人が就労（農業・製造・清掃等の福祉的就労・一般就労）に向けて「どう成長すれば働けるようになるか」を可視化・サポートするウェブアプリです。

### 対象ユーザー
- **利用者**: 軽度知的障害を持つ 18 歳以上の大人
- **支援者**: 家族・支援員・作業所スタッフ

---

## フェーズ 1（実装済み）

### 機能 1：性格・特性診断

12 問の選択式質問で以下の 6 特性を測定：

| 特性 | 説明 |
|------|------|
| 集中力 | 単純反復作業への耐性 |
| コミュニケーション力 | 指示理解・報告 |
| 体力・持続力 | 身体作業の向き不向き |
| 几帳面さ・正確性 | 製造系に重要な正確性 |
| 感情コントロール | パニック・癇癪の傾向 |
| 学習意欲・変化への適応 | 新しいことへの積極性 |

**配慮事項**
- 質問文はひらがな・やさしい日本語
- 短い文・絵文字つき
- 大きなボタンで回答

**診断結果の出力**
- 強み（2〜3 項目）
- 課題（2〜3 項目）
- 向いている仕事タイプ（以下から）
  - 🌱 農業・園芸系
  - 🔧 製造・組み立て系
  - 🧹 清掃・環境整備系
  - 🍱 食品加工系
  - 🛒 接客・販売補助系

---

### 機能 2：成長ロードマップ（就労準備ステップ）

診断結果をもとに 3 段階のロードマップを提示：

| ステップ | テーマ | 内容例 |
|---------|--------|--------|
| ステップ 1 | 生活習慣・基礎スキル | 時間を守る、挨拶、報告 |
| ステップ 2 | 作業スキル | 指示どおりに動く、繰り返し作業の練習 |
| ステップ 3 | 就労準備 | 職場体験・ジョブコーチとの連携イメージ |

各ステップに「今日できる小さな行動」を表示。

---

### 機能 3：日々の記録・振り返り

- 今日できたこと / 難しかったこと を記録（テキスト入力）
- 感情スタンプ選択（5 種類：とてもよい〜とてもつらい）
- 体調記録（3 段階：よい・ふつう・つらい）
- 1 週間の振り返りグラフ（感情推移）

---

### 機能 4：支援者（家族・支援員）ダッシュボード

- 支援対象者の日々の記録を確認
- 週間感情グラフの閲覧
- 支援者メモ機能（気になることを記録）
- AI によるアドバイス（API なしの場合はフォールバックメッセージ）

---

## 将来フェーズ（ロードマップ）

### マッチング機能
診断結果と求人情報のマッチング（農業系・製造系・清掃系・食品加工系・接客補助系の事業所情報と連携）

### コミュニケーション練習
よくある職場シーンのロールプレイ機能：
- 「わからないとき何て言う？」
- 「急にお腹が痛くなったら？」
- 「ミスをしてしまったら？」

### ルーティン管理
毎日の仕事前・仕事後のルーティンチェックリスト：
- 朝のルーティン（起床・食事・準備）
- 仕事前チェック（持ち物・体調確認）
- 仕事後チェック（日誌・振り返り）

### 成功体験の記録アルバム
「できた！」をビジュアルで残すフォトアルバム機能：
- できたことに写真やスタンプをつけて保存
- 週・月ごとに振り返れる

### 支援者間の連携ノート
家族・支援員・作業所スタッフが情報共有できる共有ノート機能：
- タグ付き記録（良かったこと・課題・連絡事項）
- 既読確認機能

## URL 設計

| URL | 機能 |
|-----|------|
| `/` | ホーム（トップページ） |
| `/diagnosis/` | 診断トップ |
| `/diagnosis/questions/` | 診断質問 |
| `/diagnosis/result/` | 診断結果 |
| `/roadmap/` | ロードマップ一覧 |
| `/roadmap/<job_type>/` | 仕事タイプ別ロードマップ |
| `/daily/` | 日々の記録ダッシュボード |
| `/daily/record/` | 今日の記録入力 |
| `/accounts/signup/` | 利用者登録 |
| `/accounts/supporter/signup/` | 支援者登録 |
| `/accounts/login/` | ログイン |
| `/accounts/dashboard/` | 支援者ダッシュボード |

## UIデザイン方針

- やさしく・わかりやすい画面
- 大きなボタン（タップしやすいサイズ）
- 短い文章・絵文字・アイコンを多用
- カラーパレット：緑・オレンジ・白を基調
- スマートフォン対応のレスポンシブデザイン
- フォントサイズは大きめ（最小 16px）

---

## 配慮事項

- OpenAI API がなくても全機能が動作するフォールバックを実装
- 診断の質問・回答はすべてひらがな・やさしい日本語
- ログインなしでも診断・ロードマップを利用可能（セッション使用）
- 支援者機能はログイン必須
